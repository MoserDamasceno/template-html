# Template HTML	
Boilerplate to start a new HTML project.
This template is made to use just html, css and js.

To run this boilerplate is needed NPM and gulp installed at machine.

To install these package please run

```
npm install -g gulp 
```

```
npm install 
bower install
gulp
gulp watch
```
